# User Management System (PHP)

This project demonstrates:
- PHP OOP concepts
- Constructors (__construct)
- Namespaces
- require_once vs include_once
- use statements with aliasing

## Folder Structure
- index.php (Entry point)
- utils/ (Classes & helpers)
- data/ (User data)

## How to Run
1. Copy project to www (WAMP)
2. Open browser
3. Run: http://localhost/user_management_OOP/index.php
